## Contest 1 Submission - Blackjack

### Usage:
1. Add `main.asm` to a Visual Studio project.
    * Or open `Project.sln` (project set on VS 2019, but can be "upgraded" to a newer version)
3. Build solution and run the resulting executable.

### Gameplay:
1. This is a singleplayer game, where the player plays against the program.
2. Use the WASD keys to traverse the pointer through the tic tac toe grid.
3. Press Space (placeholder key) to confirm selection, and place an X marker.
4. The program will make a selection and place an O marker.
5. Repeat steps 2 through 4 until a winner is determined or a tie occurs.
